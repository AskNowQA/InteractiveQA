Welcome to IQA - an Interactive Question Answering system

IQA is an interactive Semantic Question Answering (QA) system. IQA interprets users' questions expressed in a natural language in terms of SPARQL queries for the DBpedia knowledge graph. The aim of IQA is to guide the users towards the intended SPARQL query by means of interaction.

IQA demo is available at: http://iqademo.sda.tech/demo/index

Step 1:

To start an interactive question answering process of IQA, please enter a question in English in the text box and press “submit” button.

Example questions include: Who is the leader of Germany? TODO: list some questions that perform well

Step 2:

  • IQA asks questions to clarify the user intent (shown on the left). These questions are called “interaction options”.
  • The user can select “yes”, “no” or “i don’t know” to indicate if the interaction option proposed by IQA is correct.
  • IQA also suggests a SPARQL query that is the most likely interpretation of the user question (shown on the right).
  • After each user interaction, IQA updates the interaction option and the SPARQL query according to the user feedback.
